Figma School UI

Type: UI/UX Design
Tools: Figma
Duration: 3 weeks
Status: Completed

Project Overview

This project is a clean, modern interface concept for a school management platform. The focus was on clarity, accessibility, and quick navigation for students, teachers, and administrators.

The design uses a structured layout with strong visual hierarchy, soft colors, and icon-driven actions to make key tasks feel simple and fast.

Design Highlights

  • Dashboard-first layout with clear at-a-glance information
  • Color-coded sections for classes, assignments, and announcements
  • Reusable components for cards, tables, and navigation
  • Responsive layout prepared for tablet and desktop use
  • Consistent spacing and typography for better readability

Design Process

Research: Reviewed existing school portals to identify common pain points such as cluttered navigation and unclear task priority.

Wireframes: Mapped key user flows including viewing classes, checking assignments, and reading announcements.

Visual System: Built a simple design system with shared buttons, cards, tags, and icons to maintain consistency.

Key Screens

Dashboard: Quick overview of schedule, tasks, and upcoming events.

Class Page: Central hub for resources, assignments, and teacher notes.

Assignments: Filterable list with status indicators and due dates.

What I Learned

  • Designing for multiple user roles with clear priorities
  • Creating scalable components in Figma
  • Building layouts that remain readable on large data screens
  • Balancing aesthetic simplicity with functional density
