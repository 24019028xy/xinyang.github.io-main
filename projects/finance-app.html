Finance Tracker App

Type: Web Application
Tools: VS Code, Figma
Duration: 6 weeks
Status: Completed

Project Overview

The Finance Tracker App helps users manage their personal spending by organizing income, expenses, and budgets in one clean dashboard. It focuses on clarity, fast data entry, and meaningful insights that make everyday money decisions easier.

This project showcases front-end UI design, data visualization, and practical UX patterns for finance tracking.

Key Features

  • Quick add for income and expense entries
  • Budget categories with monthly limits
  • Spending charts and trend summaries
  • Search and filter by category and date
  • Responsive layout for mobile and desktop
  • Export-ready summary view

Technical Stack

Frontend: HTML5, CSS3, and JavaScript for UI, interactivity, and dynamic data views.

Design: Wireframed and prototyped in Figma to validate layout and interactions before development.

Data Visualization: Built simple charts and summary cards to highlight key trends.

Development Highlights

Clean UX Flow: Designed a minimal input flow that allows users to log expenses in a few seconds.

Visual Clarity: Focused on readability with clear categories, consistent spacing, and intuitive summaries.

What I Learned

  • Designing finance-focused user flows
  • Building responsive dashboards
  • Visual hierarchy for data-heavy screens
  • Prototyping and iterating on UX
