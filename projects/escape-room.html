Escape Room Game

Type: Game Development
Tools: Unity, C#
Duration: 6 weeks
Status: Completed

Project Overview

The Escape Room Game is an immersive puzzle-solving experience built in Unity using C#. Players navigate through mysterious rooms filled with intricate puzzles, hidden clues, and interactive objects. The game challenges players to think critically and work collaboratively to escape before time runs out.

This project showcases game development skills including scene management, physics interactions, UI implementation, and puzzle design. The game features multiple difficulty levels and replayability elements.

Key Features

  • Multiple escape room scenarios with varying difficulty levels
  • Interactive object system with physics-based interactions
  • Inventory system for collecting and using items
  • Timer-based gameplay mechanic with score tracking
  • Responsive UI with intuitive controls
  • Sound design and ambient audio feedback
  • Save/Load functionality for game progression

Technical Highlights

Game Architecture: Built using Unity's prefab system for scalable object management and scene-based architecture for efficient level design.

Programming: Developed custom C# scripts for puzzle logic, player interaction systems, and game state management. Implemented object pooling for performance optimization.

User Experience: Designed intuitive UI with clear visual feedback for puzzle completion and interactive hints system for player guidance.

Challenges & Solutions

Challenge: Managing complex puzzle logic and ensuring proper event sequencing.

Solution: Implemented event-driven programming patterns using C# delegates to handle interdependent puzzle mechanics reliably.

What I Learned

  • Advanced game development concepts in Unity
  • Object-oriented programming best practices with C#
  • Puzzle design and player engagement mechanics
  • Performance optimization techniques for games
  • Importance of playtesting and iteration
